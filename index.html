About Us

Hi! We are high schoolers at Pinewood in Los Altos and we enjoy programming. We would love to teach this to kids (ages 9-15)! We think that programming should be fun and strive to teach it that way.

Python Camp

Mon-Fri: 9AM-1PM

Target audience: kids 9-15 years old

During this camp you will learn what Python is and how to program with it! We will go over programming fundamentals (variables, loops, functions, etc) and end with a final project. No programming experience is required!

Web Development Camp

Mon-Fri: 9AM - 1PM

Target audience: kids 9-15 years old

During this camp you will learn how to make your own websites using HTML, CSS, and Javascript. We will go over programming fundamentals (variables, loops, functions, etc) and a real website will be built to show off! No programming experience is required!

iOS Development Camp

Mon-Fri: 9AM-1PM

Target audience: kids 9-15 years old

During this camp you will learn how to make your very own iOS apps using Swift and Xcode.
